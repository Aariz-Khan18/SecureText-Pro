# 🔐 SecureText Pro — Smart Text & Security Toolkit

A premium web-based toolkit for secure text operations, password generation, and cryptographic hashing with a modern and responsive UI.

🌐 **Live Demo:** https://securetext-pro.onrender.com/

---

## ✨ Highlights
- 🔐 Secure password generator (customizable)  
- 📝 Advanced text utilities (format, clean, transform)  
- 🛡️ Cryptographic hash generator (SHA-256, SHA-384, SHA-512)  
- 🌙 Dark & ☀️ Light mode support  
- ⚡ Fast, client-side processing (no data sent to server)  
- 📱 Fully responsive (Mobile + Desktop)  

---

## 🛠️ Tech Stack
- HTML5  
- Tailwind CSS  
- JavaScript (Web APIs)

## 🎯 Key Features

### 🔑 Password Generator
- Adjustable length (4–64)  
- Uppercase, lowercase, numbers, symbols  
- Option to avoid ambiguous characters  
- Secure random generation using Web Crypto API  

### 📝 Text Utilities
- Uppercase / Lowercase / Capitalize  
- Remove extra spaces  
- Reverse text  
- Word & character counter  

### 🛡️ Security Tools
- SHA-256, SHA-384, SHA-512 hashing  
- Instant client-side processing  
- Copy-to-clipboard support
